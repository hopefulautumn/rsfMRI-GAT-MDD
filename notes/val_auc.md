`val_auc` 和 `val_f1` 都是“验证集（validation set）上的指标”。

1. `val_auc`  
- 是验证集 ROC-AUC。  
- 衡量模型把正类排在负类前面的能力。  
- 取值一般在 0 到 1：  
  - 0.5 约等于随机  
  - 越接近 1 越好  
- 对类别不平衡通常更稳一些。

2. `val_f1`  
- 是验证集 F1-score。  
- 由精确率和召回率综合而来：  
  $F1 = \frac{2PR}{P+R}$  
- 更关注“查得准 + 查得全”的平衡。  
- 当你关心正类识别质量时很常用。

一句话理解：
- `val_auc` 看排序区分能力  
- `val_f1` 看在某个阈值下的分类平衡表现

所以训练日志同时看这两个，比只看一个更全面。


val_auc 的全称一般是：

Validation AUC (ROC-AUC)

中文可写作：

验证集 ROC 曲线下面积（AUC）

val_f1 的全称一般是：

Validation F1-score

中文可写作：

验证集 F1 值（F1 分数）
